Shook - share your unwanted books

Along with another NC graduate, I built a website that allows users to sign up, give their unwanted books away and explore books available for them to take, whilst being able to see how far away a book is and have the option of contacting the 'seller'.

View the demo | the front-end code | the back-end code

It was built using:

  • Vue.js
  • A variety of Amazon Web Services
  • Google Books API
  • distance matrix API

I learnt alot from this project and in hindsight, can identify better ways to implement certain functionality.
