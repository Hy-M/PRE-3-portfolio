Shook - Kirklees book share

Along with another NC graduate, I built a website that allows users to sign up, give their unwanted books away and explore books available for them to collect within the Kirklees metropolitan borough.

Due to the pandemic, this project was built entirely remotely using tools such as Discord, Trello and Slack. Building such a large project remotely with one other person was a great learning opportunity.

View the site | the front-end code

It was built using:

  • Vue.js
  • A variety of Amazon Web Services
  • Google Books API
