Garden diary app

For my first mobile development project, I built a mobile app that allows users to sign up and track their plants. It makes use of a third-party API to identify plants that users upload from their camera and camera roll. The back-end is a RESTful API that handles CRUD operations.

It was built using:

  • React Native & Expo
  • PostgreSQL, Knex & Express.js
  • plant.id API
  • AWS Cognito
