A reddit-style news app

A full-stack application displaying news about Northcoders. The data is accessed through a RESTful API with many endpoints, whilst the front-end is a single-page React application. Users can post a comment, delete their own comment, vote on articles and comments, and browse through filtered and sorted articles.

View the hosted site | the front-end code | the back-end code

The front-end was built using:

  • React
  • Reach Router
  • Axios
  • Mocha & Chai

My front-end code was split into components which are independant, reusable bits of code.

The back-end was built using:

  • PostgreSQL
  • Knex.js & node-postgres
  • Express.js
  • PostgreSQL
  • Mocha, Chai & Chai Sorted
  • Supertest

The back-end code follows a model-view-controller architecture for seperation of the applications' concerns.
